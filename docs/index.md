# Flux UI Chat

Flux UI chat interface for Laravel AI with streaming, model picker, and optional RAG.

## Features

- Streaming chat responses
- Conversation list with search, filters, and sorting
- Bookmarked messages
- Anonymous chat (no persistence)
- Optional RAG via `RagContextProvider`
- Optional cross-conversation memory via `MemoryContextProvider`
- File uploads via `ChatUploadService`
- Reactions and bookmarks (laravel-markable)
